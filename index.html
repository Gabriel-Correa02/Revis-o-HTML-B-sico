Links

Link Externos

Use a tag "a", dentro dessa tag irá aparecer o texto para que possa ser clicado. Para deixar esse link clicavel será necessário usar o atributo "href", e dentro desse atributo deverá ser inserido o link da página destino.

Google

Para que seja aberto em uma nova guia acrescente o atributo target="_blank".

Links Internos

Pode ser utilizado as mesmas tag e atributos de links Externos. Clique aqui.

Usando Imangens

Use a tag "img" + os atributos "src" + "alt" + "tittle".
Src: Usado para colocar o endereço da imagem.

Alt: Utilizado para informar oq é aquela imagem caso ela quebre.

Title: Usado para mostrar o nome da imagem quando posiciona o mouse em cima dela.

Tabela

Use a tag "table" para criar uma tabela. Nessa tag podera ser utilizado as seguintes tags:

th: cabeçalho da tabela, ou coluna.

tr: linha de uma tabela.

td: dados da tabela.

Use as tags "th" ou "td" sempre dentro de uma tr, que representa uma linha.
